# freeradiantbunny

The name of this applicaitons is freeradiantbunny.

This is an open source software projecjt.

The purpose of freeradiantbunny is to develop websites for permaculture designers.

This is implement with the Rust programming language.

The database schema is written for postgresql.

The freeradiantbunny application is a database-backed website designed as a plant list system with an herb garden information system for permaculture design method people and in fact the system may double as an multimedia inquiring system aiming to make the herb garden maps a delightful and educational experience.

Our mascot is Kitto, the botany bunny, and is said to be an expert on all things gardening and Radiant Bunny and so on if only he could be asked about it before hopping quickly away.

For more information about freeradiantbunny, see the **website**: [freeradiantbunny.org](https://freeradiantbunny.org/)
